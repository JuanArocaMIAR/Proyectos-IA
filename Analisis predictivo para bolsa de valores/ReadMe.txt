# Optimización de Modelos de Predicción de Precios de Acciones

## Descripción
Este proyecto utiliza técnicas de aprendizaje automático para predecir los precios de las acciones de Google (GOOGL). Se implementan y evalúan múltiples modelos, incluyendo Regresión Lineal, K-Vecinos más Cercanos, Árbol de Decisión, Random Forest, XGBoost y LightGBM. El objetivo es identificar cuál de estos modelos ofrece el mejor rendimiento en términos de precisión y robustez.

## Objetivos
- Evaluar el rendimiento de diferentes modelos de aprendizaje automático en la predicción de precios de acciones.
- Comparar métricas como RMSE, MAE y R² para determinar el modelo más efectivo.
- Realizar predicciones a futuro basadas en los modelos entrenados.

## Tecnologías Utilizadas
- **Python**
- **Pandas**: Para la manipulación de datos.
- **NumPy**: Para cálculos numéricos.
- **Scikit-Learn**: Para la implementación de modelos de aprendizaje automático.
- **XGBoost**: Para el modelo de boosting.
- **LightGBM**: Para un modelo eficiente de boosting.
- **Matplotlib**: Para la visualización de resultados.
- **yfinance**: Para descargar datos históricos de acciones.

## Conclusiones
Los resultados muestran que la Regresión Lineal fue el modelo más eficaz para este conjunto de datos, seguido de Random Forest y LightGBM. Este análisis proporciona una base sólida para futuras exploraciones en predicciones de precios en el mercado de valores.
